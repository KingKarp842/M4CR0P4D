# M4CR0P4D
A Macropad to help me do certain programmable actions, with 8 keys, LEDs, and an OLED screen!  It uses a Seeed Studio XIAO RP2040 as the brain, running KMK firmware, allowing each key to be mapped to shortcuts, controls, or literally anything else. This macropad is designed with a custom PCB and case, and I hope to come back and reiterate on it later.

I made this project to learn more about PCB designing and keyboard firmware, especially because I want to build a keyboard for a different project later.  I also wanted a small, customizable device that could actually help me, whether it is while CADding or just simply doing work.  (I can also flex to my friends that I built this lol)

I will source and buy all parts myself, but here is a general BOM.

* 1x Seeed Studio XIAO RP2040
* 1x Custom PCB (my KiCad design)
* 1x 0.91” OLED display (GND–VCC–SCL–SDA)
* 8x MX-style mechanical switches
* 8x Keycaps (Either bought or 3D Printed)
* 8x SK6812 MINI-E RGB LEDs
* 4x M3x12mm screws
* 4x M3 heat-set inserts
* 1x Top plate (3D Printed)
* 1x Bottom case (3D Printed)
* 1x USB-C cable (I mean you need it)
